Le but de ce projet est de réaliser une application web en utilisant NodeJs et React.
Cette Application sera le site vitrine d'un zoo. On pourra donc visiter le site en mode non connecté. Il faudra aussi prévoir un mode connecté, comportant 2 rôles distincts, le rôle visiteur et le rôle administrateur. Les textes en bleu sont optionnels, mais permettent l'excellence ;).

1- Site Vitrine / non connecté

  • Sur la page d'accueil, on dispose d'un menu et d'un ensemble de texte/image décrivant votre site (laisser place à votre imagination)
  • Le menu devra comporter un lien de connection/inscription
  • Le menu devra comporter un lien vers la liste des animaux
  • il faudra avoir une page comportant la liste des animaux présent dans le zoo
  • permettre une recherche des animaux par nom
  • il faudra pouvoir visualiser un animal en cliquant dessus
  • Proposer sur l'écran d'accueil une carte représentant le positionnement des animaux dans votre zoo

2- Connection

  • Réalisez une page de connection
  • Réalisez une page d'inscription
  • Suite à une inscription, l'utilisateur créé devra être stocké dans votre base de données, afin de lui permettre de se connecter la fois d'après.

3- Site / connecté

  • Une fois connecté, un lien supplémentaire apparaitra dans le menu "Favoris"
  • L'utilisateur pourra voir ses animaux favoris
  • Permettre à l'utilisateur de favoriser un ou des animaux

4- Sécurité

  • Il faudra mettre en place un mécanisme de token stocké en base de données
  • Ce token doit avoir une expiration

5- Administration

  • l'administrateur devra pouvoir ajouter un animal
  • l'administrateur devra pouvoir modifier un animal
  • l'administrateur devra pouvoir supprimer un animal
  • Déconnecter un utilisateur
