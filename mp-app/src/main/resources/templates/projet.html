Le but de ce projet est de réaliser une application web en utilisant NodeJs et React.
Cette Application permettra la gestion d'un recueil de citation. Il y aura 2 points d'entrée, à cette application. Le site web, qui vous permettra de consulter, rechercher, créer des citations. Le bot discord, qui vous permettra de la meme manière de consulter les citations, d'en créer. Les textes en bleu sont optionnels, mais permettent l'excellence ;).

1- Site web / non connecté

  • Sur la page d'accueil, on dispose d'un menu et d'un ensemble de texte/image décrivant votre site (laisser place à votre imagination)
  • Le menu devra comporter un lien de connection
  • Le menu devra comporter un lien vers la liste des citations
  • il faudra avoir une page comportant la liste des citations
  • permettre une recherche des citations par contenu et/ou par auteur
  • il faudra pouvoir visualiser une citation en cliquant dessus, et ainsi avoir une fiche contenant la citation, l'auteur et la date d'ajout

2- Connection

  • Utiliser l'api discord afin de gérer la connection

3- Site web / connecté

  • Une fois connecté, un lien supplémentaire apparaitra dans le menu "Ajouter"
  • L'utilisateur pourra alors ajouter une citation
  • Permettre à l'utilisateur de favoriser une ou des citations
  • Permettre à l'utilisateur de consulter ses citations favorisées

4- BOT discord

  • Vous devrez créer un bot discord, qui vous permettra d'effectuer les operations suivantes
  • L'utilisateur pourra ajouter une citation
  • l'utilisateur pourra consulter les citations
  • l'utilisateur pourra consulter les citations d'une personne
  • Permettre à l'utilisateur de consulter ses citations favorisées
  • Le bot sera ajouter sur un serveur que je vous fournirez ultérieurement.
  • Vos commande devront être préfixé par un identifiant unique, permettant d'éviter les doublons (ie: /trd-list-citation)
  • il faudra dans le readme me fournir la liste des commandes ;)

4- Sécurité

  • Il faudra mettre en place un mécanisme de token stocké en base de données
  • Ce token doit avoir une expiration

5- Administration

  • Il y aura un rôle administrateur, vous devrez donc gérer un schéma vous permettant de savoir si un utilisateur est admin.
  • l'administrateur devra pouvoir ajouter une citation
  • l'administrateur devra pouvoir supprimer une citation
  • Déconnecter un utilisateur
