Le but de ce projet est de réaliser une application web en utilisant nodejs et react. Cette application gérera des événements découpés en manches. Il faudra donc permettre aux utilisateurs de se connecter/s'inscrire, sinon quoi il ne sauront qu'en simple consultation. On pourra ainsi consulter ce planning, et venir s'y inscrire. Vous pourrez en option faire un mode administrateur.

1- Planning / non connecté

  • Sur la page d'accueil, on observe la liste des plannings/événements
  • Il faut pouvoir visualiser un planning, celui ci est représenté par différentes manches, chaque manche possède un nom, et on peut inscrire autant de participants que l'on souhaite par manche.

2- Connection

  • Réalisez une page de connection
  • Réalisez une page d'inscription
  • Suite à une inscription, l'utilisateur créé devra être stocké dans votre base de données, afin de lui permettre de se connecter la fois d'après.

3- Planning / connecté

  • Une fois connecté, vous devrez pouvoir visualiser le planning, et vous inscrire aux différentes manches

4- Sécurité

  • Il faudra mettre en place un mécanisme de token stocké en base de données
  • Ce token doit avoir une expiration

5- Administration

  • l'administrateur devra pouvoir créer un planning
  • inscrire une personne existante
  • Déconnecter un utilisateur
